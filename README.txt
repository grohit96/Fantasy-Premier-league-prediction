Code is in a .ipynb file Jupyter Notebook, developed on VSCode

Required packages: jupyter, numpy, pandas, scikit-learn
pip commands needed:
    pip install jupyter
    pip install numpy
    pip install pandas
    pip install scikit-learn

Select 'Run All' button at the top of VSCode once the cs584_semester_project.ipynb file has been opened in the CODE directory, a file called model_output.csv will be created
showing the team made up of players with the highest predicted scores from the model for each gameweek, along with the points they actually scored.
Note: the dataset is included in the zip file, no need to download it for the code to run.
Second Note: Runtime is approximately 8-10 minutes
